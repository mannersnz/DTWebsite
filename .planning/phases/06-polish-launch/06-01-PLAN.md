  <files>src/layouts/MainLayout.astro</files>
  <action>
    Update MainLayout.astro head section to add SEO essentials:

    **Add to head section:**

    1. **Open Graph tags** (for social sharing):
       ```html
       <meta property="og:title" content={title} />
       <meta property="og:description" content={description} />
       <meta property="og:type" content="website" />
       <meta property="og:locale" content="en_NZ" />
       <meta property="og:site_name" content="Digital Tempo" />
       ```

    2. **Twitter Card** (for Twitter/X sharing):
       ```html
       <meta name="twitter:card" content="summary" />
       <meta name="twitter:title" content={title} />
       <meta name="twitter:description" content={description} />
       ```

    3. **Additional SEO tags:**
       ```html
       <meta name="robots" content="index, follow" />
       <meta name="author" content="Digital Tempo" />
       <link rel="canonical" href="https://digitaltempo.nz/" />
       ```

    4. **Theme color** (for mobile browsers):
       ```html
       <meta name="theme-color" content="#1e40af" />
       ```

    Keep existing tags (charset, viewport, description, title, generator).
  </action>
  <verify>npm run build succeeds, view page source shows OG tags</verify>
  <done>MainLayout includes Open Graph, Twitter Card, and SEO meta tags</done>
